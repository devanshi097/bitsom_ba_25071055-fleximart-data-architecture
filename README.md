# FlexiMart Data Architecture Project





This repository contains the complete data architecture solution for the FlexiMart case study.



* &nbsp;**Project Structure**



\- data 

&nbsp; Raw input datasets including customers, products, and sales data.



\- part1-database-etl/

&nbsp; ETL pipeline implementation, schema documentation, business SQL queries, and data quality report.



\- part2-nosql/ 

&nbsp; NoSQL (MongoDB) analysis, product catalog JSON, and MongoDB operations.



\- part3-datawarehouse/

&nbsp; Data warehouse star schema design, warehouse SQL scripts, and analytical queries.



* &nbsp;**Technologies Used**

\- Python

\- SQL

\- MongoDB

\- Git \& GitHub



* &nbsp;**How to Run**

\- ETL scripts are located in `part1-database-etl`

\- SQL queries are provided for analysis and reporting

\- MongoDB operations are documented in Part 2
